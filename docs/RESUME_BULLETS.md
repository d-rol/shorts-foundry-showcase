# Resume And Interview Bullets

## Resume-friendly bullets
- Built a private-code orchestration pipeline that converts long-form episodic video into ranked short-form highlight candidates and vertical-ready clips.
- Designed a deterministic selector plus human-in-the-loop review workflow instead of relying on a fully opaque end-to-end model.
- Implemented subtitle, smart-crop, packaging, and calibration layers to support repeatable short-form clip production.
- Created selector audit and review artifacts to improve ranking quality using real editorial feedback rather than subjective tuning alone.
- Validated the system across multiple long-form source profiles, including dialogue-heavy and shorter comedy material.

## Interview framing
Good phrasing:
- I optimized for a strong shortlist, not fake full autonomy.
- I treated review as a first-class product layer, not as an admission of failure.
- I used AI where it added leverage and deterministic logic where repeatability mattered more.
- I built the system around artifacts and calibration loops so it could actually improve over time.

## What To Emphasize
- orchestration thinking;
- iterative validation;
- practical product tradeoffs;
- ability to ship useful systems under imperfect conditions.
