# Roadmap

## Current Stage
The project is beyond prototype stage but not yet a fully autonomous editorial system.

## Next Steps
### 1. Shortlist-aware AI rerank
Use AI on `top-10` candidates only, not as the raw selector.

### 2. Stronger review calibration
Continue using labeled review sheets to improve selector behavior with real editorial feedback.

### 3. Packaging refinement
Expand lightweight template switching into clearer visual families.

### 4. Better productization
Improve operator UX, review tooling, and showcase readiness.

## Long-Term Direction
The strongest long-term model is likely:
- deterministic shortlist generation;
- AI-assisted ranking inside the shortlist;
- human-in-the-loop final decision when needed.
