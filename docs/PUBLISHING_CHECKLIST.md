# Publishing Checklist

Before publishing the private showcase repo:
- replace all `TBD` items in `README.md`
- add 2-5 screenshots to `media/screenshots/`
- add a demo video link in `README.md`
- confirm no private code exists in the repo
- confirm no copyrighted source media exists in the repo
- confirm no secrets or local config files exist in the repo
- verify README renders correctly on GitHub
