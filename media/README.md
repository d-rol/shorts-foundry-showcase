# Media Placeholders

Put public-safe showcase assets here before publishing:
- screenshots in `media/screenshots/`
- demo clips or walkthrough assets in `media/video/`

Do not add:
- copyrighted source episodes
- private generated outputs tied to non-public media
- any internal code screenshots that reveal implementation you want to keep private

Recommended exact assets: see `docs/MEDIA_SHOT_LIST.md`.
